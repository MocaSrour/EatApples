# EatApples
## How to run
```cmd
git clone https://github.com/MocaSrour/EatApples
cd EatApples
javac -d dest *.java
cd dest
java SnakeGame
```
## Requirements
  Eat Apples game written in Java so you need a Java Development Kit (JDK).
  You can download a JDK from [here](https://www.oracle.com/java/technologies/downloads/)
